Install
------------------------------------------------------------------------------

``learn_dvc`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-dvc

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-dvc
